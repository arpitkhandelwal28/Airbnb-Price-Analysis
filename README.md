# 🏠 Airbnb Price Analysis using Python & Power BI

## 📌 Project Overview

This project analyzes Airbnb listing data to uncover pricing trends, availability patterns, host performance, and neighborhood insights. The analysis was performed using Python for data cleaning, exploratory data analysis (EDA), and business insights, while Power BI was used to build an interactive dashboard for visualization and decision-making.

The project aims to help hosts, investors, and stakeholders understand market trends and optimize pricing strategies.

---

## 🎯 Objectives

- Analyze Airbnb pricing trends across neighborhoods and room types.
- Identify top-performing hosts based on customer reviews.
- Explore availability patterns across room categories.
- Compare average prices across boroughs.
- Track revenue trends over time.
- Generate actionable business recommendations.

## 💡 Key Insights

- Manhattan has the highest average Airbnb price.
- Entire home/apartment listings dominate the market.
- Top hosts receive significantly more reviews than others.
- Average pricing differs considerably across boroughs.
- Brooklyn and Manhattan have the highest listing concentration.
- Availability varies significantly across room types.

## 📌 Business Recommendations

- Optimize pricing strategies based on neighborhood demand.
- Focus on high-demand neighborhoods for maximum revenue.
- Encourage hosts to maintain high guest satisfaction to increase reviews.
- Improve listing availability during peak seasons.
- Use competitive pricing for better occupancy rates.
